# Ultimate Custom Hybrid Keyboard

A modern, fast, and customizable Android keyboard built with Kotlin. This keyboard features a Material You design, drag-and-drop key remapping, and smart gesture support.

## üöÄ Features
- **Material You Theme**: Beautiful light and dark themes based on Google's design language.
- **Drag & Drop**: Manually remap your keys by dragging them (Edit Mode).
- **Spacebar Trackpad**: Precise cursor control by sliding on the spacebar.
- **Smart Delete**: Gesture-based word deletion.
- **Privacy Focus**: Automatic incognito theme for private sessions.

## ‚å®Ô∏è How to Enable
Once installed, follow these steps to start using the keyboard:
1. **Launch the app**: Open "Ultimate Keyboard" from your app drawer.
2. **Enable in Settings**: Tap the "Enable Keyboard" button. This will take you to Android's "Manage Keyboards" screen. Toggle on **Ultimate Custom Keyboard**.
3. **Select as Default**: Tap the "Select Keyboard" button in the app and choose **Ultimate Custom Keyboard** from the list.
4. **Start Typing**: Open any app (like WhatsApp or Notes) and enjoy your new keyboard!
